# Codex + CC Switch 账号管理教程

这篇教程主要讲如何使用CC Switch管理 Codex账号及账号切换

## 下载CC Switch

下载地址：https://github.com/farion1231/cc-switch/releases



## 使用CC Switch配置Codex账号

### 以中转站为例

> 任何中转站都可以，这里随便找了一个野鸡中转，不推荐

### 中转站提供的信息

- Api key
- Api 地址

![](/images/articles/codex-cc-switch/api-info.png)

### 配置CC Switch
> 选择 codex 点击添加账号
![](/images/articles/codex-cc-switch/add-account.png)

> 选择自定义配置
![](/images/articles/codex-cc-switch/custom-config.png)

> 填写账号信息
![](/images/articles/codex-cc-switch/account-form.png)

> 点击添加按钮

> 对刚添加的账号进行启用
![](/images/articles/codex-cc-switch/enable-account.png)
